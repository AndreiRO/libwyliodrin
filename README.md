libwyliodrin
============


Requirements
------------
Libwyliodrin has the following requirements
  * redis-server
  * libhiredis
  * libjansson

Raspberry Pi
  * wiringPi

ArduinoGalileo
  * libmaa

Languages
  * swig

Python
  * distools

Javascript
  * distools


Install
-------

    make $BOARDNAME - can be raspberrypi or arduinogalileo
    sudo make install
